# 💊 ChronoPharMark

**Τοπική εφαρμογή διαχείρισης φαρμακείου** — σάρωση barcode (GS1 DataMatrix),
παρακολούθηση λήξεων, εκτύπωση ετικετών. **Όλα τα δεδομένα μένουν στον
υπολογιστή σας** — τίποτα δεν αποθηκεύεται στο διαδίκτυο.

> Αυτό το repository περιέχει **μόνο το έτοιμο πρόγραμμα** (installer/.exe).
> Ο πηγαίος κώδικας είναι ιδιωτικός.

---

## ⬇️ Λήψη & Εγκατάσταση

1. Κατεβάστε το **`ChronoPharMark.exe`** από την τελευταία έκδοση:
   👉 **[Releases](../../releases/latest)**
2. Τρέξτε το. Δεν χρειάζεται εγκατάσταση — είναι αυτόνομο.
3. (Για εκτύπωση ετικετών) εγκαταστήστε **Ghostscript**:
   https://www.ghostscript.com/releases/gsdnld.html

> Τα Windows ίσως δείξουν προειδοποίηση SmartScreen για άγνωστο εκδότη →
> «Περισσότερες πληροφορίες» → «Εκτέλεση παρ' όλα αυτά».

---

## ✨ Δυνατότητες

- **Σάρωση** GS1 DataMatrix (Product Code, Serial, Lot, Λήξη) με πιστόλι barcode.
- **Καταστάσεις φαρμάκων** με χρώματα: 🟢 Εντάξει · 🟡 Λήξη σύντομα · 🔴 Ληγμένα.
- **Φίλτρα & περίοδοι** (τελευταίες ημέρες/τρίμηνο/προσαρμογή), ταξινόμηση,
  αναζήτηση.
- **Εκτύπωση ετικετών** 50×30mm με barcode.
- **Backup / Εισαγωγή / Εξαγωγή** (CSV, Excel).
- **Αυτόματες ενημερώσεις** με ένα κλικ.

---

## 🔑 Δοκιμή & Άδεια

- Στην πρώτη εκτέλεση ξεκινά **δωρεάν δοκιμή 14 ημερών** με πλήρη λειτουργία.
- Μετά τη δοκιμή χρειάζεται **κλειδί ενεργοποίησης**. Μέσα στην εφαρμογή θα δείτε
  τον **Κωδικό Μηχανήματος** — στείλτε τον μας για να σας δώσουμε κλειδί.
